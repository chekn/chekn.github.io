
# chrome小插件 - 找工作小助手
本插件是 找离自己最近 工作的小作品，功能单一，效用可人。 
软件把工作 在地图上 很直观的 显示出来。厌烦 海投,想找钱多对口离家近的 专职高工 可以拿来试试 简单高效

### 安装很简单:
1. 扩展页面中，点添加

2. 点击 小公文包 图标，弹出界面。界面上方弹出  浏览器定位 点允许，就可以 定位你的位置，展示搜索功能了

3. 输入 要找的 工作 <xx高级工程师> ，再点 找工作
